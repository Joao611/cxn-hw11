# Define function for iterative page rank update
update.x <- function(x, alpha, beta) {
  out.degree <- rowSums(M)
  
  # D_inv <- diag(1 / out.degree)
  # x <- alpha * x %*% D_inv %*% M + beta
  
  for (i in seq_along(x)) {
    tmp <- 0
    for (j in seq(1, nrow(M))) {
      if (i != j && M[j, i] != 0){
        tmp <- tmp + x[j] / out.degree[j]
      }
    }
    x[i] <- alpha * tmp + beta
  }

  return(x)
}

page.rank <- function(n.iter, alpha, M, G, draw.plot = FALSE) {
  beta <- (1 - alpha) / nrow(M)
  x <- rep(0, 11)
  
  for (i in seq(n.iter)) {
    x.old <- x
    x <- update.x(x.old, alpha, beta)
    if (draw.plot) {
      print(paste('iteration', i, sqrt(sum((x-x.old)^2))))
      plot(G, vertex.size = x*5e2, vertex.color=smileycolors, layout=l)
    }
  }
  return(x)
}

It can be seen in the first graph that having a negative alpha (-0.01) makes previously small nodes bigger, which means nodes with a small rank now obtain a higher rank. Decreasing the negative alpha further, as is the case in the second graph with alpha = -0.1, the same effect is observed with the oiginally small nodes growing even bigger but now itâs also observed that the originally big nodes get smaller, so what used to be the more important nodes are now lower ranked.

This means the algorithm is reversed, attributing a higher rank to less important nodes. This is explained by the fact that alpha is a factor applied to each nodeâs strength given by its neighbors, so a negative alpha will result in higher values for inferior neighbor strengths.

Page rank produces proper results for alpha values lower than 1. At alpha = 1, the nodesâ scores are all 0 and so the vertices do not appear in the plot because their sizes are also 0. This is because the initial score for each node is 0, and the only way to increase it when all the neighbors are also 0 is the (1 - alpha) / |V| parcel in the formula. When alpha = 1, this factor will be 0 and therefore the scores canât increase. Furthermore, alpha > 1 would lead to invalid plots in this case because the scores would be negative.

It should be noted the transition around this boundary is smooth, with the nodesâ scores lowering gradually as alpha increases up to 1.
